# Welfare AI Assistant

AI-based welfare service recommendation and case management assistant.

AI 기반 복지 사각지대 발굴 및 맞춤형 복지서비스 연계 플랫폼입니다.  
복지 대상자의 상황을 분석하여 위기 유형을 분류하고, 적합한 복지서비스를 추천하며, 상담 요약과 사례관리 기록 작성을 지원하는 것을 목표로 합니다.

---

## 📌 Project Overview

현대 사회복지 현장에서는 다양한 복지제도가 존재하지만, 대상자가 본인에게 맞는 제도를 직접 찾기 어렵고 담당자의 상담·서류·사례관리 업무 부담도 큽니다.

또한 위기가구는 문제가 심각해진 이후에 발견되는 경우가 많으며, 소득·주거·건강·돌봄·고립 문제가 복합적으로 얽혀 있어 단순 키워드 검색만으로는 적절한 지원 방안을 찾기 어렵습니다.

본 프로젝트는 이러한 문제를 해결하기 위해 AI를 활용하여 다음 흐름을 지원합니다.

> 복지 사각지대 발굴 → 초기상담 → 맞춤형 서비스 추천 → 사례관리 지원

---

## 🚩 Problem

- 복지제도가 많아도 대상자가 본인에게 맞는 제도를 알기 어려움
- 담당 공무원과 사회복지사의 상담·서류·사례관리 업무가 과중함
- 위기가구는 문제가 커진 뒤 발견되는 경우가 많음
- 소득·주거·건강·돌봄·고립 문제가 복합적으로 얽혀 있어 단순 키워드 검색으로 적절한 지원을 찾기 어려움

---

## 🧩 Main Features

### M1. 위기가구 위험 신호 분석 모델

행정 데이터, 상담 기록, 신청 이력, 공공요금 체납, 고립 위험 신호 등을 기반으로 복지 사각지대 가능성을 점수화합니다.

### M2. AI 초기상담 및 상담요약 모델

복지 대상자의 상황을 자연어로 입력받고, AI가 필요한 추가 질문을 생성합니다.  
상담 이후에는 주요 내용을 요약하여 담당자의 기록 업무를 지원합니다.

### M3. 맞춤형 복지서비스 추천 모델

사용자의 상황을 기반으로 신청 가능한 복지제도를 추천합니다.  
예를 들어 생계, 주거, 의료, 돌봄, 고용 관련 복지서비스를 상황에 맞게 제안합니다.

### M4. 사례관리 지원 대시보드

사회복지사나 담당자가 대상자별 상황, 위기 유형, 추천 서비스, 상담 요약, 후속 조치 이력을 한눈에 확인할 수 있도록 지원합니다.

---

## 🔄 Service Flow

```text
[대상자 정보 입력]
        ↓
[AI 초기상담 / 추가 질문 생성]
        ↓
[위기 유형 분류]
생계 / 주거 / 의료 / 돌봄 / 고립 / 고용
        ↓
[복지서비스 추천]
긴급복지, 주거급여, 의료비 지원, 돌봄서비스 등
        ↓
[상담 요약 및 사례관리 기록 자동 생성]
        ↓
[담당자 검토 후 지원 연계]
